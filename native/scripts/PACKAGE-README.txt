Runescape Clicker for Windows 11 x64

1. Extract this package to a local folder.
2. Run Install-RunescapeClicker.ps1 from PowerShell.
3. The installer copies the self-contained app to %LocalAppData%\RunescapeClicker by default.
4. A Start Menu shortcut named "Runescape Clicker" is created for the current user.

Notes
- This package is Windows-only and x64-only.
- The app targets Windows 11 build 26100 or newer.
- If Windows blocks automation against an elevated target app, run both apps at the same privilege level.
