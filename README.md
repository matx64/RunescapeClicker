# Runescape Clicker

An open source Runescape Clicker with graphical interface developed in Python.

![img](./static/main.png)

## Features

- Left or Right Click on desired position
- Keyboard Press (1, 2, 3, Space)
- Set Delay
- Set automatic stop after x seconds

## Tech

- Python3
- Pip Packages:
  - Tkinter
  - Pynput
  - bindglobal
